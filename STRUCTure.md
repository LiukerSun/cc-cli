# CC-CLI 项目结构

```
cc-cli/
├── bin/
│   └── cc                # 主脚本（核心功能)
├── docs/
│   ├── configuration.md   # 配置文档
│   ├── usage.md            # 使用指南
│   └── troubleshooting.md # 故障排除
├── tests/
│   └── test.sh             # 测试脚本
├── .gitignore              # Git 忽略文件
├── CHANGELOG.md             # 更新日志
├── CONTRIBUTING.md        # 贡献指南
├── LICENSE                 # MIT 许可证
├── README.md               # 项目主文档
├── install.sh               # 安装脚本
├── config.example.json     # 示例配置
└── Makefile               # Make 构自动构建

```

## 文件说明

- **bin/cc** - 主脚本，包含所有核心功能
- **docs/** - 项目文档
  - `configuration.md` - 配置指南
  - `usage.md` - 使用示例
  - `troubleshooting.md` - 故障排除指南
- **tests/** - 测试目录
  - `test.sh` - 测试脚本
- **install.sh** - 一键安装脚本
- **README.md** - 项目文档
- **LICENSE** - MIT 许可证
- **.gitignore** - Git 忽略文件
- **config.example.json** - 示例配置

- **CHANGELOG.md** - 版本更新日志
- **CONtribution.md** - 贡献指南
- **Makefile** - Make 构自动构建
